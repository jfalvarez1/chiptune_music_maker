# Chiptune Music Maker

A music creation tool focused on producing authentic chiptune and 8-bit style sounds.

## Overview

Chiptune Music Maker is designed to help musicians and enthusiasts create retro-style music reminiscent of classic video game soundtracks. The project aims to emulate the distinctive sounds of vintage sound chips like the NES's 2A03, Game Boy's LR35902, and Commodore 64's SID chip.

## Features

- **Classic Waveforms**: Generate pulse, triangle, sawtooth, and noise waves
- **Channel Limitations**: Authentic constraints inspired by vintage hardware
- **Pattern Sequencer**: Compose music using pattern-based sequencing
- **Instrument Editor**: Create and customize chiptune instruments
- **Export Options**: Save your creations in various audio formats

## Sound Chip Emulation

The project aims to support sound profiles inspired by:

| Chip | System | Channels |
|------|--------|----------|
| 2A03 | NES | 2 pulse, 1 triangle, 1 noise, 1 DPCM |
| LR35902 | Game Boy | 2 pulse, 1 wave, 1 noise |
| SID | Commodore 64 | 3 voices with multiple waveforms |
| AY-3-8910 | Various | 3 square wave channels |
